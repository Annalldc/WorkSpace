TCC IOS ⭐

A minha ideia de TCC é criar um site web voltado para animais em extinção e como podemos ajuda-los, por exemplo, nesta página teríamos uma parte específica para doação
com o intuito de ajudar esses animais e juntamente com os profissionais que atuam junto nesta missão conseguir reverter a situação atual destes animais que sofrem devido a ação humana.
